中国历史地图集软件（谭其骧）  
=======================

设备要求：  
---------
  安卓5.1以上  
  winXP/win7/win10  
  MAC(OSX10.10以上)    

主要功能：  
---------
  `多个历史时期的高清地图（分幅图拼接），全部离线查看。`  
  `春秋战国、秦汉、三国、西晋、隋唐宋和元明清支持地点搜索并定位。`  
  `朝代切换后视图位置不变，方便查看历史沿革。`  
  `支持卫星和地形地图叠加，方便查看基本地形。`  
  `支持离线检索《中国历史地名大辞典》（史为乐），方便了解地名沿革详情。`  
  `支持添加多种地图标注，如手绘涂鸦、箭头、文字、折线等，支持测距。`
